# 商品标签识别规范：format + WordArt

## 1. 目标

对商品正面印刷图案输出两个标签：

1. `format`：版式标签，只输出固定标签标题。
2. `WordArt`：艺术字标签，只输出图片中实际识别到的艺术字文案。

最终只输出以下两个字段，不输出编号、解释、置信度或其他字段。

```json
{
  "format": "文案-上下结构",
  "WordArt": "HAPPY BIRTHDAY"
}
```

---

## 2. 固定输出格式

```json
{
  "format": "<固定版式标签>",
  "WordArt": "<提取到的艺术字原文>"
}
```

### 2.1 字段规则

#### `format`

- 只输出一个固定标签标题。
- 不输出分类编号。
- 不输出判断原因。
- 不输出数组。
- 不自行创造新标签。

正确：

```json
{
  "format": "文案-居中元素散排",
  "WordArt": "HAPPY BIRTHDAY"
}
```

错误：

```json
{
  "format": {
    "code": "2.1.1",
    "name": "文案-居中元素散排"
  }
}
```

#### `WordArt`

- 只提取图片中具有艺术字特征的具体文字。
- 保持原图中的文字内容和阅读顺序。
- 不输出 `true`、`false`。
- 不输出艺术字类型。
- 不输出判断原因。
- 不翻译，不改写，不纠正原图拼写。
- 多行文字按正常阅读顺序合并，用一个半角空格分隔。
- 若有多组彼此独立的艺术字文案，用 ` / ` 分隔。
- 若没有艺术字，输出空字符串 `""`。

有艺术字：

```json
{
  "format": "文案-简约图文",
  "WordArt": "MY 21ST RODEO"
}
```

无艺术字：

```json
{
  "format": "图案-自由式散排",
  "WordArt": ""
}
```

---

# 3. 分析范围

只分析商品主要正面印刷区域。

忽略以下内容：

- 水印；
- 商品展示页文字；
- 图片边缘被遮挡或重复的盘面；
- 盘子侧面重复印刷；
- 商品阴影；
- 白色展示背景；
- 尺寸、价格、货号等商品信息；
- 非主设计内容中的路牌文字、店铺招牌、建筑标识、交通标识、Logo、门牌等。

## 3.1 文案类判断原则

不能仅因为图片中出现文字，就判为文案类。

只有当文字承担以下作用之一时，才进入文案类：

- 是商品主设计文案；
- 是主题表达的核心内容；
- 是画面主要视觉内容之一；
- 对整体版式结构有明确影响。

场景里的路牌、招牌、建筑标识、交通标识、Logo、门牌等，如果只是场景组成，不进入文案类。

---

# 4. `format` 固定标签

## 4.1 图案类

固定标签：

- `图案-自由式散排`
- `图案-主体式散排`
- `图案-居中元素类`
- `图案-居中场景类`
- `图案-环绕中心实物`
- `图案-环绕中心空白`
- `图案-单图案款`
- `图案-条纹/纯色款`

---

## 4.2 文案类

固定标签：

- `文案-居中元素散排`
- `文案-居中上散排下场景`
- `文案-居中整块背景/场景`
- `文案-居左/右整块背景或组合`
- `文案-简约图文`
- `文案-上下结构`
- `文案-文字环绕中心场景/元素`
- `文案-元素环绕文字居中`

---

# 5. 图案类判断规范

## 5.1 图案-自由式散排

多个元素自由、随机地分布在版面中，没有明确的中心主体。

主要特征：

- 各元素相对平等；
- 大小、角度、间距可以不同；
- 不形成完整中心场景；
- 不形成规则环绕；
- 没有一个元素明显主导整个画面。

---

## 5.2 图案-主体式散排

存在一个明确主体，其他小元素自由散排并作为辅助装饰。

主要特征：

- 有一个最大、最突出或最重要的主体；
- 其他元素明显从属于主体；
- 辅助元素未形成规则环绕；
- 主体可以居中，也可以略微偏移。

与自由式散排的区别：

- 无明确主角：`图案-自由式散排`
- 有明确主角：`图案-主体式散排`

---

## 5.3 图案-居中元素类

存在清晰的“主体 + 背景”关系，主体集中放置在背景之上，四周有明显留白或背景空间。

主要特征：

- 中心是一个独立元素或一组紧密组合元素；
- 主体边界清楚；
- 主体没有直接铺满整个商品表面；
- 四周存在明显背景空间；
- 不构成完整场景。

---

## 5.4 图案-居中场景类

完整场景集中位于版面中心，并与周围背景形成清晰区分。

主要特征：

- 中心包含多个具有情节或空间关系的元素；
- 人物、动物、建筑、植物或物件共同组成完整画面；
- 场景主体与背景之间存在明显留白或背景空间；
- 不能简单看作一个独立元素。

---

## 5.5 图案-环绕中心实物

外围元素沿圆形、弧形或边缘围绕中心实物或中心场景。

主要特征：

- 中心有明确实物、人物、动物、图标或场景；
- 外围元素形成规则或较明显的环绕关系；
- 中心与外围有清晰层级。

---

## 5.6 图案-环绕中心空白

外围元素形成圆形、弧形或边框式环绕，但中心保留明显空白。

主要特征：

- 中心没有主体；
- 中心区域明显留白；
- 装饰集中在外围；
- 环绕结构清晰。

---

## 5.7 图案-单图案款

主体本身就是整个版面，单个完整图案被放大并直接铺满商品表面，不存在独立的“主体 + 背景”关系。

主要特征：

- 完整图案占满或基本占满整个印刷区域；
- 主体与版面融为一体；
- 四周没有明显留白或独立背景空间；
- 不要求是重复小图案；
- 单个完整图案放大铺满也属于本类。

例如：

- 澳大利亚国旗完整铺满整个盘面；
- 一张完整地图直接覆盖整个商品表面；
- 单个大型图案扩展至边缘并成为整个版面。

与居中元素类的区别：

- 有独立主体、四周有背景空间：`图案-居中元素类`
- 图案本身直接成为整个版面：`图案-单图案款`

---

## 5.8 图案-条纹/纯色款

以基础颜色或规则纹理铺满整个版面。

包括：

- 纯色；
- 渐变色；
- 横条纹；
- 竖条纹；
- 斜条纹；
- 波浪条纹；
- 格纹；
- 棋盘格；
- 规则几何底纹。

---

# 6. 文案类判断规范

## 6.1 文案-上下结构（最高优先）

当主文案与一个可以独立成立的主体形成清晰的上下阅读关系时，优先输出此标签。

结构一：

```text
文案
↓
主体
```

结构二：

```text
主体
↓
文案
```

主体必须能够脱离背景独立成立，例如：

- 人物；
- 动物；
- 汽车；
- 蛋糕；
- 城市主体；
- 礼物；
- Pie；
- 牛仔；
- 游戏手柄；
- 其他明确物件或角色。

背景不能作为主体。

即使文案位于画面中心，只要文案和独立主体形成明确上下结构，也优先判为：

`文案-上下结构`

---

## 6.2 文案-文字环绕中心场景/元素

主文字沿圆形、弧形或外围路径围绕中心主体。

主要特征：

- 中心有明确场景或元素；
- 文字处于外围；
- 文字沿弧线、圆周或边缘排列；
- 文字形成包围或半包围关系。

---

## 6.3 文案-元素环绕文字居中

主文案位于中心，外围元素形成规则环绕。

主要特征：

- 中心是文字；
- 元素沿圆周、边缘或封闭路径排列；
- 环绕关系规则、清晰；
- 不属于自由散排。

与居中元素散排的区别：

- 外围形成规则环绕：`文案-元素环绕文字居中`
- 外围元素自由分布：`文案-居中元素散排`

---

## 6.4 文案-居中上散排下场景

主文案位于中心区域，上方有自由散排元素，下方形成完整场景或明确主体组合。

主要特征：

- 上方视觉较轻、较散；
- 下方视觉较重；
- 下方不是单个简单主体，而是场景或组合结构；
- 整体呈“上散排—中文案—下场景”关系。

---

## 6.5 文案-居左/右整块背景或组合

主文案明显位于左侧或右侧，另一侧由完整主体、组合元素或整块背景平衡画面。

主要特征：

- 文字不在视觉中心；
- 文字与另一侧内容形成左右结构；
- 另一侧内容可以独立形成主体或组合；
- 不是简单上下结构。

---

## 6.6 文案-居中整块背景/场景

文案与完整背景或场景融合，不存在“文案 ↓ 独立主体”或“独立主体 ↓ 文案”的关系。

主要特征：

- 文案叠加在连续背景或完整场景上；
- 背景是整体画面的一部分；
- 背景不能被视为一个独立主体；
- 文案与背景共同形成主视觉。

例如：

- Miami；
- Australia；
- 67；
- ALL I SEA IS LOVE；
- Galentine's Gang；
- Merry Christmas（格纹）；
- LEVEL 10。

注意：

格纹、宇宙、海洋、城市环境、游戏背景等若作为整块连续背景，应判为本类，而不是上下结构。

---

## 6.7 文案-简约图文

画面以文案为主要内容，只搭配极少装饰。

主要特征：

- 没有独立主体；
- 没有完整场景；
- 没有多个自由散排元素；
- 没有规则环绕；
- 装饰通常只是少量星星、线条、爱心、色块或简单符号。

---

## 6.8 文案-居中元素散排（默认文案版式）

主文案位于中心，多个独立元素在周围自由散排。

主要特征：

- 文案是视觉中心；
- 周围存在多个元素；
- 元素的位置、大小、方向较自由；
- 不形成规则环绕；
- 不形成文案与独立主体的明确上下关系；
- 不存在连续完整背景或场景。

当图片属于文案类，但不符合更高优先级标签时，默认考虑：

`文案-居中元素散排`

---

# 7. 文案类判断优先级

按以下顺序判断：

1. `文案-上下结构`
2. `文案-文字环绕中心场景/元素`
3. `文案-元素环绕文字居中`
4. `文案-居中上散排下场景`
5. `文案-居左/右整块背景或组合`
6. `文案-居中整块背景/场景`
7. `文案-简约图文`
8. `文案-居中元素散排`

注意：

- `文案-上下结构`为最高优先级。
- `文案-居中元素散排`为默认文案版式。
- 不得仅根据文字是否居中直接判断。
- 必须先识别文字、独立主体、连续背景、散排元素和环绕关系。

---

# 8. `WordArt` 艺术字识别规范

## 8.1 目标

`WordArt`不是艺术字类型标签，而是图片中艺术字具体内容的提取字段。

正确：

```json
{
  "WordArt": "HAPPY BIRTHDAY"
}
```

错误：

```json
{
  "WordArt": true
}
```

错误：

```json
{
  "WordArt": "像素游戏字"
}
```

---

## 8.2 哪些文字属于艺术字

文字具有明显视觉设计，且文字造型本身是商品设计的一部分。

包括但不限于：

- 手写字；
- 连笔花体字；
- 笔刷字；
- 卡通圆体字；
- 气泡字或膨胀字；
- 像素字；
- 游戏风字体；
- 西部牛仔字体；
- 复古展示字；
- 哥特字；
- 明显描边字；
- 空心字；
- 立体字；
- 阴影字；
- 渐变字；
- 图案填充字；
- 多色字；
- 字母与动物、花朵、皇冠、火焰、猫爪、车轮等元素融合；
- 使用图形替代字母或字符；
- 明显波浪、扭曲、拉伸或变形字体；
- 通过多种字体混排形成明显艺术效果。

---

## 8.3 哪些文字不提取到 `WordArt`

以下情况单独出现时，不视为艺术字：

- 普通黑体；
- 普通宋体；
- 普通无衬线字体；
- 普通常规衬线字体；
- 仅改变大小；
- 仅加粗；
- 仅斜体；
- 仅改变颜色；
- 普通居中；
- 普通换行；
- 普通全大写；
- 非主设计文案；
- 水印；
- 商品信息；
- 场景中的招牌、路牌、Logo或门牌。

---

## 8.4 提取范围

只提取具有艺术字特征的那部分主文案。

例如，图片中：

- `Cosmetology School`使用手写艺术字；
- `SURVIVOR`使用普通常规字体。

则输出：

```json
{
  "WordArt": "Cosmetology School"
}
```

不要因为同一画面中存在普通文字，就把所有文字全部提取。

如果整句主文案都属于艺术字：

```json
{
  "WordArt": "HAPPY BIRTHDAY"
}
```

如果两组独立艺术字都需要提取：

```json
{
  "WordArt": "OUT / I'M 10"
}
```

如果没有艺术字：

```json
{
  "WordArt": ""
}
```

# 9. 完整判断流程

## 第一步：确定商品正面主印刷区域

排除水印、叠放商品、侧面、阴影和展示背景。

## 第二步：判断是否存在主设计文案

- 有主设计文案：进入文案类。
- 没有主设计文案：进入图案类。
- 场景文字、Logo、路牌等非主设计文字不改变分类。

## 第三步：识别版式结构

判断：

- 是否存在独立主体；
- 是否形成上下结构；
- 是否形成规则环绕；
- 是否存在连续完整背景或场景；
- 是否为左右结构；
- 是否只有少量装饰；
- 是否为自由散排；
- 图案是否直接铺满整个版面。

## 第四步：输出 `format`

只能从固定标签中选一个，并只输出标题名。

## 第五步：识别艺术字

逐段检查主文案的字体造型，只提取真正具有艺术字特征的文字。

## 第六步：输出最终JSON

只保留：

```json
{
  "format": "...",
  "WordArt": "..."
}
```

# 11. 强制规则

1. 最终只能输出合法JSON。
2. JSON中只能包含`format`和`WordArt`两个字段。
3. `format`只能输出一个固定标签标题。
4. `WordArt`只能输出图片中实际存在的艺术字原文。
5. `WordArt`不得输出布尔值。
6. `WordArt`不得输出艺术字类型。
7. 不得输出版式编号。
8. 不得输出解释、理由、置信度或备注。
9. 不得把水印、商品信息或场景标识当作主文案。
10. 不得仅因出现文字就进入文案类。
11. `文案-上下结构`为文案类最高优先级。
12. `文案-居中元素散排`为文案类默认标签。
13. 背景不能被当作上下结构中的独立主体。
14. 单个完整图案放大铺满整个版面，属于`图案-单图案款`，不属于`图案-居中元素类`。
15. 没有艺术字时，`WordArt`必须输出空字符串`""`。
